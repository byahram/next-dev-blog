| 함수 이름                     | 설명                                                        | 사용 위치                                          |
| ----------------------------- | ----------------------------------------------------------- | -------------------------------------------------- |
| getAllPosts()                 | 모든 카테고리의 포스트를 불러오고 날짜 기준 최신순으로 정렬 | 전체 포스트 리스트 페이지, 검색 등                 |
| getPostSlugs()                | 각 포스트의 [category]/[slug] 조합을 반환                   | generateStaticParams()에서 정적 페이지 생성에 사용 |
| getPostsByCategory(category)  | 지정된 카테고리의 포스트만 최신순으로 반환                  | /blog/[category] 페이지                            |
| getPostBySlug(category, slug) | 특정 포스트의 내용을 읽어 frontmatter와 content 반환        | /[category]/[slug] 페이지 (실제 글 렌더링)         |
| getPostsByTag(tag)            | 주어진 태그를 포함한 포스트 목록 반환                       | 태그 필터링 시                                     |
| searchPosts(query)            | title, description, tags에 query가 포함된 포스트 필터링     | 검색 기능에서 사용                                 |
